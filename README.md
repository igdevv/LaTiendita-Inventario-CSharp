El sistema "La Tiendita" es una aplicación de consola desarrollada en C# que permite gestionar un inventario de productos.

El programa utiliza arreglos paralelos para almacenar nombres, precios y cantidades de los productos.

Las funcionalidades principales incluyen registrar productos, listar inventario, actualizar el stock, eliminar artículos y generar facturas de venta.

El sistema está estructurado mediante métodos para organizar cada funcionalidad, utilizando estructuras de control como if, while, for y switch.
